Detailed Description

Structure storing the result of a single flash lidar ray casting event

  • double range between emitting pixel and intersection point (nan if no intersection found)
  • std::pair<int,int> pixel_indices Pair formed with the row index (first) and column index (second) of the emitting pixel
  • arma::vec start_N Inertial frame coordinates of the emitting pixel
  • arma::vec end_N Inertial coordinates of the first intersection point between the ray and the target. Set to arma::vec(fill::nan) if no intersection is found
